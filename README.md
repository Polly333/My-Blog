# A personal blog website using Django

## Features:

I can :
- Log in and log out
- Add a new blog
- Edit , delete a blog
- Save a blog as draft and publish it later
- Reply to comments from reader 

Readers can :
- View the whole list of blogs and each blog
- Can comment on blogs 
- View others' comments
